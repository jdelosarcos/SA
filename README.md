# Student Degree Outcome Prediction Dashboard

This is the lightweight Streamlit deployment package. It avoids Colab commands and heavy libraries that may cause cloud deployment to load indefinitely.

## Files to upload to GitHub
- app.py
- preprocessing.py
- train_models.py
- requirements.txt
- runtime.txt
- packages.txt
- README.md
- data/ folder optional
- models/ folder optional

## Streamlit/AuthKit setting
Main file path: `app.py`

## Use
Upload your Excel dataset in the sidebar, then use the modules.
